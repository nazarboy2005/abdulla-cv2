# E-Portfolio

## Setup Instructions

### Adding Project Files

Place your project PDF files in the `public/projects/` folder:

1. `public/projects/cv.pdf` - Your CV/Resume
2. `public/projects/logistics-supply-chain.pdf` - Logistics and Supply Chain Project
3. `public/projects/iced-arabic-coffee.pdf` - Iced Arabic Coffee Marketing Plan

These files will be automatically accessible for download through the portfolio interface.

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view your portfolio.
